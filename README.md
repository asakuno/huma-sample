# Huma API プロジェクト

このプロジェクトは、Go言語とHumaフレームワークを使用したRESTful APIです。
モジュール化されたアーキテクチャにより、拡張性と保守性を重視した設計になっています。

## アーキテクチャ概要
- **フレームワーク**: Huma v2 (OpenAPI準拠)
- **ルーター**: Chi
- **データベース**: MySQL 8.0
- **開発環境**: Docker Compose + Air (ホットリロード)

## プロジェクト構造
- `cmd/` - アプリケーションエントリーポイント
- `app/` - アプリケーション層
- `pkg/` - 再利用可能なパッケージ
- `shared/` - 共通ユーティリティ
- `scripts/` - 運用スクリプト
- `.docker/` - Docker設定
